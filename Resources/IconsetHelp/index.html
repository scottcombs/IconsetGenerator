Iconset Generator Help

Welcome to the Iconset Generator help page. Iconset Generator is a small development tool which generates .iconset folders from a PNG file. One high-rez image is used to generate all the various sizes Apple demands when building an application. You can spend more time on the design of your application icon and less time using an image editor to make all the images.

How this is done is through two command-line applications, sips and iconutil. Iconset Generator takes the one high-res PNG and scales it to the proper resolutions inside an .iconset folder. Then Iconset Generator can build an ICNS file from the .iconset using the application, iconutil. For you it is seamless and easy. Just drag your PNG file into the imageview and Iconset Generator automatically performs the conversion.
